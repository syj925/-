# 校园墙管理后台使用说明

> **版本**：v1.0.0  
> **更新时间**：2025-06-28  
> **状态**：开发中（认证系统已完成）

## 📋 系统概述

校园墙管理后台是一个基于Vue3 + Element Plus的现代化管理系统，用于管理校园墙平台的用户、内容、话题等各项功能。

## 🚀 快速开始

### 环境要求

- **Node.js**：v16.0.0 或更高版本
- **npm**：v7.0.0 或更高版本
- **浏览器**：Chrome 88+、Firefox 85+、Safari 14+

### 启动步骤

1. **启动后端服务**
   ```bash
   cd server
   npm run dev
   ```
   - 服务地址：http://localhost:3000
   - API文档：http://localhost:3000/api-docs（待实现）

2. **启动管理后台**
   ```bash
   cd admin
   npm run dev
   ```
   - 访问地址：http://localhost:8888

3. **登录管理后台**
   - 用户名：`admin`
   - 密码：`admin123`

## 🔐 登录说明

### 管理员账号

| 字段 | 值 | 说明 |
|------|----|----|
| **用户名** | `admin` | 系统管理员账号 |
| **密码** | `admin123` | 默认密码，建议首次登录后修改 |
| **角色** | `admin` | 管理员权限 |
| **邮箱** | `admin@campus-wall.com` | 管理员邮箱 |

### 登录流程

1. 访问 http://localhost:8888
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后自动跳转到仪表盘

### 权限说明

- ✅ **管理员权限**：可以访问所有管理功能
- ❌ **普通用户**：无法访问管理后台
- ❌ **未登录用户**：自动跳转到登录页

## 🔧 技术架构

### 前端技术栈

- **框架**：Vue 3.5.13
- **UI库**：Element Plus 2.9.10
- **状态管理**：Pinia 3.0.2
- **路由**：Vue Router 4.5.1
- **HTTP客户端**：Axios 1.9.0
- **图表库**：ECharts 5.4.3
- **构建工具**：Vite 6.3.5

### 后端技术栈

- **框架**：Express.js
- **数据库**：MySQL + Sequelize ORM
- **缓存**：Redis
- **认证**：JWT
- **日志**：Winston
- **文档**：Swagger（计划中）

## 🐛 故障排除

### 常见问题

1. **无法登录**
   - 检查用户名密码是否正确
   - 确认后端服务是否启动（http://localhost:3000）
   - 查看浏览器控制台是否有错误信息

2. **页面空白**
   - 检查前端服务是否启动（http://localhost:8888）
   - 清除浏览器缓存
   - 检查网络连接

3. **API调用失败**
   - 确认后端服务正常运行
   - 检查API地址配置（admin/src/utils/api.js）
   - 查看网络请求是否被拦截

### 日志查看

- **前端日志**：浏览器开发者工具 Console 标签页
- **后端日志**：server终端输出
- **网络请求**：浏览器开发者工具 Network 标签页
